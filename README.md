# OpenCV Event Demo

## Getting Started

1. 安装Python（推荐3.10以后的版本）

2. 下载本仓库代码
    ```shell
    git clone https://github.com/RoggeOhta/OpenCV-Event-Demo.git
    cd OpenCV-Event-Demo
    ```
   
3. 安装相关第三方库
   ```shell
   pip install -r reqirements.txt
   ```

4. 运行测试，排除简单错误
5. 运行`main.py`

## Memo

1. 关于测试的详细说明请参考`docs/doc.md`
